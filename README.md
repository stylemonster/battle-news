# ⚔️ 战场新闻资讯中心

> 实时抓取国外新闻源的战争/冲突资讯，自动翻译、分类、生成网站

## 功能特点

- 📰 **多源抓取**: Reuters, BBC, CNN, Al Jazeera, NYT, Twitter, Reddit
- 🌍 **多战区覆盖**: 俄乌战场、中东局势、伊朗、以色列等
- 🈶 **自动翻译**: 英文新闻自动翻译为中文
- 🏷️ **智能分类**: 按战区、国家自动打标签
- 📈 **热度分析**: 统计各战区新闻热度
- 🌐 **网站生成**: 自动生成静态网站

## 快速开始

```bash
# 安装依赖
npm install

# 运行一次
node scripts/scraper.js --once

# 持续运行（每30分钟）
node scripts/scraper.js
```

## 定时任务

可在 OpenClaw 中设置 cron 任务自动运行：

```bash
# 每小时运行一次
node battle-news/scripts/scraper.js --once
```
